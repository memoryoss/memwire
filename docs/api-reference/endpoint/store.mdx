---
title: 'Store memory'
description: "Add messages to a user's memory graph"
icon: 'database'
---

`POST /v1/memory`

### Request body

| Field | Type | Required | Description |
|---|---|---|---|
| `user_id` | string | ✅ | User to store memory for |
| `messages` | array | ✅ | List of `{role, content}` message objects |
| `app_id` | string | | Namespace memories per application |
| `workspace_id` | string | | Namespace memories per workspace or team |
| `agent_id` | string | | Namespace memories per agent |

### Example

```bash
curl -X POST http://localhost:8000/v1/memory \
  -H "Content-Type: application/json" \
  -d '{
    "user_id": "alice",
    "app_id": "app_a",
    "workspace_id": "team_1",
    "messages": [
      { "role": "user", "content": "I prefer dark mode and short answers." }
    ]
  }'
```

### Response

```json
{
  "stored": 1,
  "memory_ids": ["mem_3f7a1c2d9e4b"]
}
```
